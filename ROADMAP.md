# Roadmap for LearnCards

## Current Milestones
- **Version 1.0**:
  - Basic flashcard creation and review system.
  - Drawing and customization functionality.
  - Progress tracking and feedback.

## Future Plans
- **Version 2.0**:
  - Integration of spaced repetition algorithm.
  - Support for multiple project types (text, images, audio).
  - More interactive drawing tools and collaborative features.
  - Enhanced analytics for learning patterns.

## Beyond 2.0
- Cloud synchronization of flashcards and drawings.
- User-friendly UI improvements and accessibility features.
- Community-driven content contributions and feedback systems.
