# Contributing to LearnCards

We welcome contributions from the community! Whether you're fixing bugs, improving documentation, or adding new features, your input is appreciated.

## Ways to Contribute

- **Bug reports**: If you encounter any issues, please report them. Be specific about the problem and steps to reproduce.
- **Feature requests**: Suggest new features that could enhance the app.
- **Code contributions**: Fork the repository, make your changes, and submit a pull request.

## Guidelines

- **Code Style**: Follow Android coding standards and best practices.
- **Pull Requests**: Ensure your code is tested and follows the existing code structure.
- **Documentation**: If you add or modify functionality, update relevant documentation.

## Reporting Issues

If you find a bug or have a feature request, please create an issue in the [Issues](https://github.com/yourusername/LearnCards/issues) section.

## Code of Conduct

Please adhere to our [Code of Conduct](CODE_OF_CONDUCT.md) while contributing.
